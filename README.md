# Bike Sales Dashboard using Microsoft Excel

## Project Overview

This project demonstrates how raw customer sales data can be cleaned, transformed, and visualized using Microsoft Excel.

The dashboard enables users to analyze customer purchasing behavior based on demographic and lifestyle factors through interactive slicers and pivot charts.

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Cleaning
- Excel Functions

## Insights

- Customers with shorter commute distances tend to purchase more bikes.
- Male customers generally have a higher average income than female customers.
- Bike purchases vary across different age groups.
- Region, education, and marital status influence purchasing patterns.
